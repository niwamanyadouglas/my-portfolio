Automated Data Cleaning with Python

This project demonstrates my ability to automate repetitive data cleaning tasks using Python. The script performs the following tasks:

  • Standardizes column names (converts to lowercase, replaces spaces with underscores).
  • Removes duplicate rows.
  • Fills missing numeric values with the mean.
  • Drops columns with excessive missing values (threshold: 50%).
  • Exports the cleaned dataset to a CSV file.

By automating these tasks, the script reduces manual effort by approximately 50%, making it a valuable tool for data analysts and researchers.

Tools Used

  • Python
  • Pandas Library
